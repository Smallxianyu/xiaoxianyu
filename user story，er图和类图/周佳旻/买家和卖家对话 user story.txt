user story

card：搜索商品
conversation：作为已注册用户或者游客在搜索栏里输入搜索物品，
              在下面就能找到所要购买的商品，
              以便顾客能够更方便地购买商品和有好的购物体验。
confirmation：模糊搜索 通过
              精确搜索 通过
              已注册用户搜索 通过
              游客搜索 通过
              搜索存在的商品 通过
              搜索不存在的商品 不通过


card：上传商品
conversation：作为已注册用户，可以在上传页面上传自己的商品简介和价格，
              以便买方在搜索页面搜索到卖方想卖的商品。
confirmation：已注册用户上传 通过
              未注册用户上传 不提供
              上传图片 通过
              上传文字说明 通过
              未上传图片 通过
              未上传文字 不通过



card：买家和卖家对话
conversation：作为买家，想要进一步了解商品的相关信息，可以在单个商品介绍页面进入对话框和卖家对话。
confirmation：已注册用户 通过
              未注册用户 不通过
              对话框中输入对话 通过
              对话框中输入图片 通过
